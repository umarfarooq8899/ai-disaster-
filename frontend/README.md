# AI Disaster Relief Frontend (React + Vite + Tailwind)

This is a frontend scaffold for the AI Disaster Relief System.

## What's included
- React + Vite
- Tailwind CSS
- React Router
- React-Leaflet map component
- Mock API using localStorage and utils/mockData
- Role-based dashboards (public, volunteer, ngo, rescue, admin)

## Run locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Notes
- This is a frontend-only scaffold. Replace `src/api/*.js` with real endpoints when backend is available.
- Map uses OpenStreetMap tiles via react-leaflet.
